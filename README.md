# LexiPulse

An optimization data processing engine that runs raw text sanitization and computational frequency matrices within an inline single-file Django infrastructure.

🔗 **[Live Production Web Application Link](PASTE_YOUR_RENDER_URL_HERE)**

## 🛠️ Tech Stack
* **Core Logic Engine:** Python / Django Web Gateway Interface
* **Algorithmic Utilities:** Regular Expressions (`re`) and `collections.Counter`
* **Interface UI Stack:** Tailwind Compiler / HTML5 Data Tables
* **Production Distribution Web Server:** Gunicorn
